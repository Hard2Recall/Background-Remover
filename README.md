# SIMPLE IMAGE BACKGROUND REMOVER (using withoutbg)

**SETUP**   
in your terminal type `pip install withoutbg` and wait for it to install    

**USAGE**   
Put image that you want backrgound removed from in the folder, then write images name and file type in `removerV1.py`.  
After that just run the programm and it will create new picture called `output` and thats it.
